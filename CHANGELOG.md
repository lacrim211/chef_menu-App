# Changelog

## 1.0.0 - Initial release
- Added core app navigation (Main, NewMeal, GuestMenu).
- Implemented FoodContext for state management (add, remove meals, compute averages).
- MainScreen: list meals, remove meals, show course averages.
- NewMealScreen: form to add meals with name, price and course (picker).
- GuestMenuScreen: filter meals by course for guests.
- Added Expo + React Native configuration files.
